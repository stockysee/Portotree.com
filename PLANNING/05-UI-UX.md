# PortoTree — UI / UX Design System

**Version:** 1.0  
**Project:** PortoTree

---

# 1. Design Philosophy

PortoTree mengutamakan pengalaman pengguna yang sederhana, cepat, dan profesional.

Prinsip utama:

- Clean
- Modern
- Premium
- Minimal
- Fast
- Consistent
- Mobile First

Inspirasi:

- Apple
- Linear
- Notion
- Vercel
- Stripe Dashboard

---

# 2. Design Principles

- Fokus pada konten.
- Minim distraksi.
- Konsisten di semua halaman.
- Navigasi mudah dipahami.
- Maksimal 3 klik menuju fitur utama.
- Gunakan whitespace secukupnya.
