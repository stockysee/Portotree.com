# 23. MVP Definition

PortoTree dianggap siap diluncurkan jika sudah memenuhi poin berikut:

- ✅ User dapat membuat akun.
- ✅ User dapat membuat resume.
- ✅ User dapat membuat portfolio.
- ✅ User dapat mengelola pengalaman, pendidikan, skill, project, dan sertifikat.
- ✅ User memiliki Public Profile.
- ✅ Resume dapat diunduh sebagai PDF.
- ✅ Sistem pembayaran premium berjalan.
- ✅ Website responsif dan SEO-friendly.

# 25. Final Goal

**PortoTree bukan sekadar Resume Builder.**

Tujuan akhirnya adalah menjadi **Career Identity Platform** tempat seseorang membangun, mengelola, dan membagikan seluruh identitas profesionalnya dalam satu akun yang dapat digunakan sepanjang perjalanan kariernya.
